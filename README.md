#  Sales Data Pipeline & Dashboard

Ein einfaches Data Engineering-Projekt mit Python, SQLite und Streamlit.  
Ziel ist es, Verkaufsdaten zu verarbeiten, zu analysieren und in einem Dashboard darzustellen.
